## Используемые технологии

- HTML — семантическая структура страницы  
- CSS — адаптивная вёрстка и стилизация  
- Flexbox и Grid Layout — построение сеток  
- БЭМ — структура и именование классов  
- Медиа-запросы — адаптация под разные разрешения экранов  

---

## Описание проекта

Одностраничный сайт о путешествиях по России.  
Проект выполнен в рамках **Яндекс Практикума** и посвящён адаптивной вёрстке под различные устройства.

На странице представлены:
- Информация о популярных направлениях в России;
- Галерея фотографий;
- Карточки мест с описанием и ссылками;
- Адаптация под десктоп, планшет и мобильные устройства.

## Как запустить проект локально

1. Клонируйте репозиторий:
```bash
git clone https://github.com/maryamgab/frontend_yandex_third_sprint.git
cd frontend_yandex_third_sprint
```

2. Запуск:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```
